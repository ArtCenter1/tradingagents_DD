# Run Report Schema

> Reference for every field in `full_states_log_{date}.json`

Generated by `TradingAgentsGraph._log_state()` in `tradingagents/graph/trading_graph.py`.

## Top-level Fields

| Field | Type | Description |
|-------|------|-------------|
| `company_of_interest` | string | Company name or ticker |
| `trade_date` | string | Date of analysis |
| `market_report` | string | Market Analyst markdown report |
| `sentiment_report` | string | Sentiment Analyst markdown report |
| `news_report` | string | News Analyst markdown report |
| `fundamentals_report` | string | Fundamentals Analyst markdown report |
| `investment_debate_state` | object | Research team's debate state |
| `trader_investment_decision` | string | Trader's investment plan |
| `risk_debate_state` | object | Risk team's debate state |
| `investment_plan` | string | Portfolio Manager's final plan |
| `final_trade_decision` | string | Final decision (BUY/SELL/HOLD + rationale) |

## `investment_debate_state`

| Field | Type | Description |
|-------|------|-------------|
| `bull_history` | string | Bull researcher's reasoning (markdown) |
| `bear_history` | string | Bear researcher's reasoning (markdown) |
| `history` | string | Debate conversation log |
| `current_response` | string | Latest debate round response |
| `judge_decision` | string | Research Manager's synthesis |

## `risk_debate_state`

| Field | Type | Description |
|-------|------|-------------|
| `aggressive_history` | string | Aggressive risk analyst's view |
| `conservative_history` | string | Conservative risk analyst's view |
| `neutral_history` | string | Neutral risk analyst's view |
| `history` | string | Risk debate conversation log |
| `judge_decision` | string | Risk Manager's final risk assessment |

## Which Panel Uses Which Fields

| Panel | Fields |
|-------|--------|
| Analysts | `market_report`, `sentiment_report`, `news_report`, `fundamentals_report` |
| Research | `investment_debate_state.bull_history`, `.bear_history`, `.judge_decision` |
| Trading | `trader_investment_decision` |
| Risk | `risk_debate_state.aggressive_history`, `.conservative_history`, `.neutral_history`, `.judge_decision` |
| Portfolio | `investment_plan`, `final_trade_decision` |
