- **Decision**: Place project as sibling to TradingAgents clone (not inside it).
  - Rationale: Clean separation; TradingAgents `git pull` works without conflict.

- **Decision**: Phase 1 = Flask instant MVP.
  - Rationale: `python app.py` → browser, zero npm, zero build. Operators see a working UI in 5 minutes.
  - Contrast: FastAPI + React requires `npm install`, Vite build, ~2h setup before any visual result.

- **Decision**: React-Grid-Layout for drag-and-drop.
  - Rationale: Mature, native `draggableHandle` support, layout serialization to localStorage, responsive breakpoints. Alternatives (dnd-kit, gridstack) require significantly more custom work.

- **Decision**: Zero modifications to TradingAgents upstream.
  - Rationale: Read-only integration via existing JSON log files. `git clone` stays vanilla. Every future TradingAgents update is automatically compatible. Fork divergence risk eliminated.

- **Decision**: Design tokens → `DESIGN.md` (Google `design.md` spec).
  - Rationale: Exportable to Tailwind/DTCG JSON later. WCAG contrast checking available via `npx @google/design.md lint`.
